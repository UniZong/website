VR Technical Artist

What you will have to do

• Collaborate with producer, programmers and artists to ensure the game is both artistically attractive and technically efficient；

• Help producer estimate assets preparation time while evaluating technical standards and restrictions with programmers;

• Write detailed assets preparation summary and completion tracking sheet;

• Test assets delivered by artists inside game project and provide accurate revising direction to artist if any needed;

• Understand technical feedback from programmers once assets tested not applicable inside the game and make corresponding quick revision according to project's needs;

• Optimize game levels based on knowledge of topology, draw calls, batching, rendering, setpass calls, shader, occlusion culling, level of details etc;

• Justify if rig and animation done in an appropriate way fitting game levels;

• Create appropriate UI strategies both artistically attractive and technically workable;

• Solve problems related to shader, particle effects, lighting.

• In-depth and constructive knowledge about optimization;

• Experience in Maya, Blender, 3DMax, Photoshop or similar;

• Experience with version control systems (Git);

• Experience with task trackers (Jira);

• Basic programming skills in Unity (C#);

• Experience in writing and editing shaders;

• Experience in game development companies;

• Availability of their own game projects;
